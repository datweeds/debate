# Profile

## Main Profile Panel
This is where you can chane your personal details and preferences.

### Profile Navigation Buttons
At the top you can Log Out, which logs you out of the system and takes you back to the page you came from. The Close button also returns you to the previous page but keeps you logged in. If you are logged out you cannot participate in debates and cannot view parts of the platform.

### Handle
This is the monilker that is visible to everyone using the platform. It should be 6 characters long at least and must be unique. If you change your handle it will change throughout the system, i.e. any debates you have participated in will automatically pick up your new handle. 

### Email
This must be a vaid email and is used only for the Debate Report team and platform to communicate with you, e.g. password reset, newsletters, and any automated alerts you may have. You can reset your password using the button here, which starts the reset workflow. You should receive an email inviting you to select a new password.

### Full Name
It is not mandatory to record this in the platform. If you do record it in your profile we will use it in system emails and newsletters, but will never reveal it to other users. If you opt to have your biography and name publically available then your name will be revealed to other users.

### Biography
You can optionally record a short biography of yourself. This will only be revealed to other users if you set it to be publically available (below). If set to public you can see it in the chamber detail panel.

### Biography and Name publicly available?
Tick this is you want your Full Name and Biography to be available to platform users. If unticked these attributes will be kept private. 

### Subscribe to the Newsletter? 
Tick this is you wish to receive the occassional newsletter by email. Untick if not, we will not be offended! However the newsletter is a good source of tips and ideas, debate theory, and updates on what's hot in the chamber.

### Button 'Update Details'
Click to save any changes you have made to profile attributes. Any validation will be applied at this point and you may receive an error message which you can correct and re-try. 

## Debate Forum Subscriptions
This panel lists all the debate forums you are subscribed to. Normally you will see House Public Forum, which is where all the public debates are hosted. If you have created or subscribed to any other private forums these will also be listed.

### Forum Name
Tells people what the forum is generally about. Set in the dashboard 'my forums' area.

### My Role
The roles you can have are: Voter (allowing you to vote and view votes), Debater (allowing you to create debate statements and host family debates), or Moderator (allowing you to create debates in the public forum and to create private debates). 

You can have different roles in different forums. For example, you can be Moderator for forums you have created and Debater or Voter in the public forum and in other private forums. Each role type has a different subscription fee which is collected by Stripe payments processing. You can change the role you have at any time. If you do change role your role will change immediately but your subscription fee will change when it renews. There are no refund provided if you downgrade.

### Change
Use this button the change your role or find out how to cancel your subscription

### Joined
This is the date your subscription to the forum started

### Status (Role)
Set to 'active' if you can vote/debate, or inactive if you have paused or deleted the subscription

### Posts
The number of statements you have created, if a Debater or Moderator.

### Payment Plan
This tells you how much you pay each month or year

### Status (Payments)
Indicates the status of your payment subscripotion workflow. Should show 'complete' if your payment was successfully processed.

### Stripe Sub_Id
Your subscription Id from Stripe, which you may need when communicating with Stripe

## Button 'Upgrade to full Public Forum access'
If you have an account but have not yet subscribed to the public forum, e.g. if you subscribed to the newsletter, then you can upgrade to Voter, Debater or Moderator using this button. It will take you through the necessary workflow.
